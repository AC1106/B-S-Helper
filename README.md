##MaxProfit 股票交易算法
這個 C# 程式用於計算在給定交易費用情況下，股票交易的最大利潤。它會識別最佳的買入和賣出日期，以實現最大利潤。
該程式使用動態規劃方法解決在有交易費的情況下，如何買賣股票以獲得最大利潤的問題。MaxProfit 函數接收一個股票價格數組和交易費作為輸入，返回一個包含最佳買入日、最佳賣出日和最大利潤的元組。


##安裝
要運行此程式，請確保您的計算機上已安裝 .NET SDK。您可以從 .NET 官方網站 下載。


##使用方法
克隆倉庫或下載源代碼。
在包含源代碼的目錄中打開終端或命令提示符。


##範例
給定股票價格 [7, 1, 5, 3, 6, 4] 和交易費 1，程式將輸出：
makefile
複製程式碼
最佳買入時間: 第 2 天, 最佳賣出時間: 第 5 天, 最大利潤: 4
這意味著最佳買入日是第 2 天，最佳賣出日是第 5 天，最大利潤為 4。


##函數
MaxProfit
csharp
複製程式碼
public static (int, int, int) MaxProfit(int[] prices, int fee)
參數
prices: 一個整數數組，表示不同日期的股票價格。
fee: 一個整數，表示每次交易的費用。
返回值
一個元組 (int, int, int)，包含：
最佳買入日 (buyDay)。
最佳賣出日 (sellDay)。
最大利潤 (maxProfit)。
Main
Main 方法初始化股票價格和交易費，調用 MaxProfit 函數並打印結果。
